## 📖 About SkyCast

**SkyCast** is a modern Android weather application that provides live weather information based on the user's current location or a searched city.

It displays temperature, forecasts, humidity, wind, pressure, sunrise, sunset and other useful weather details using a Weather API.

---

## ✨ Main Features

### 📍 Current Location Weather
Automatically detects the user's location and displays live weather details.

### 🌡️ Complete Weather Information
Shows temperature, weather condition, feels-like temperature, humidity, wind speed and air pressure.

### 🕐 Hourly Forecast
Displays temperature and weather changes throughout the day.

### 📅 Three-Day Forecast
Provides weather predictions for the upcoming three days.

### 🌅 Sunrise and Sunset
Shows the daily sunrise and sunset times for the selected location.

### 🔍 Search Any City
Search for any city and view its current weather and forecast.

## 🔊 Voice Weather Greeting

SkyCast can automatically speak the current weather when the application opens.

> “Good afternoon! In Ahmedabad, it is 34°C with sunny weather.”

The voice greeting can be turned **ON or OFF** from the application settings.

---

## 🔔 Weather Alerts

Users can receive notifications for important weather conditions:

- High temperature
- Strong winds
- High humidity
- Low air pressure

Weather alerts can also be turned **ON or OFF**.
