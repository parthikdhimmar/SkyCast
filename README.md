## 📖 About SkyCast

**SkyCast** is a modern weather application that provides live and detailed weather information.

The app can automatically detect the user's current location or allow the user to search for any city. It displays the current temperature, hourly forecast, upcoming weather, sunrise and sunset times, wind information, humidity, pressure and other useful weather details.

SkyCast also includes voice greetings, weather alerts, a map weather view and a home-screen widget.

## 🔌 Weather API

SkyCast uses a Weather API to provide live and accurate weather information.

The API is used to retrieve:

- Current weather
- Hourly forecast
- Three-day forecast
- Sunrise and sunset times
- Wind information
- Humidity
- Pressure
- Visibility
- UV index
- Precipitation details

> [!IMPORTANT]
> Never upload your private Weather API key directly to GitHub.  
> Store it securely in a local configuration file and add that file to `.gitignore`.
